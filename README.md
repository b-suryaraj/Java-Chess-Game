<h1>♟️ Java Chess Game ♟️</h1>
Welcome to Java Chess Game! 🎮 This is a fully functional chess game built using Java and Swing, featuring an intuitive GUI and exciting gameplay mechanics.

<h3>✨ Features</h3>
✅ Complete Chess Board with all standard pieces 🏰 <br>
✅ Legal Moves Highlighting (Green for valid, Red for captures) 🔍
<br>✅ Undo & Redo Moves for better control 🔄
<br>✅ Move History to track past moves 📜
<br>✅ Check & Checkmate Detection ⚠️
<br>✅ Automatic Turn Switching ♻️
<br>✅ Simple, Clean UI 🎨

<h3>🚀 Technologies Used</h3>
Java ☕<br>
Swing (Javax.Swing) 🎨<br>
Object-Oriented Programming (OOP) 📦<br>
<h3>🔧 How to Play?</h3>
1️⃣ Select a piece to highlight valid moves.<br>
2️⃣ Click a highlighted square to move.<br>
3️⃣ Checkmate the opponent's king to win! 👑<br>
4️⃣ Use Undo/Redo buttons to correct mistakes.<br>

<h3>🌟 Future Improvements</h3>
🔹 AI Opponent 🤖<br>
🔹 Multiplayer Mode 🌍<br>
🔹 Timer & Scoring System ⏳<br>

<h3>🎯 How to Run?</h3>
Clone this repo and run ChessBoard.java in any Java-supported IDE (VS Code, IntelliJ, Eclipse).
